# Python Basics: Math, Type Casting, and String Operations

This repository contains practice exercises and notes focused on the core fundamentals of Python programming. It is designed to track progress in understanding basic data types, arithmetic operations, and essential string manipulation methods.

---

## Key Topics Covered

### 1. Basic Arithmetic and Variables
* Learning how Python handles fundamental math operations like addition, subtraction, and multiplication.
* Practicing dynamic variable assignment, allowing values stored inside variables to be updated and manipulated throughout a script.

### 2. Type Casting and Data Security
* Understanding explicit data type conversion, specifically changing integers into strings to print them cleanly within sentences.
* Exploring how to safely extract numerical data from text strings to prevent program crashes (Value Errors) when dealing with messy input data.

### 3. String Manipulation (Split and Join)
* **The Split Method:** Using Python's built-in tools to chop a single string into a list of individual pieces based on a specific separator (such as spaces, commas, or hyphens).
* **The Join Method:** Reversing the split process by taking a list of separate words or items and gluing them back together into a single cohesive string using a designated connector.
